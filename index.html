The final implementation of our version of model 1 can be found in alignBackFW.py this python script includes 2 versions of the algrithm, one translating from English to French and one from French to English. These 2 versions are used in combination to extract more data out of the supplied corpus/bitext.
Our best combination(first iteration convergence was achieved) of forward(E given F) and backwards(F given E) is 13 iterations of 1 itr forward and 1 itr backward.

We had run into 2 main snags along the way:
  • We had missed the part about implementing the decoding where we need to print out the file. The version with this flaw interestingly gave us the same answer as the one with the correct mapping if run for enough iteration ~2000

  • The other issue was that we called python's set() instead of iterating over every element in the list, it removed duplicates. This decreased the acuracy of our results by roughly 10 AER
